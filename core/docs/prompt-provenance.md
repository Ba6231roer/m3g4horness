# Prompt Provenance — mgh-sast

Every prompt file in `core/prompts/` is ported from `vvaharness/` (Visa, Inc. /
Project Glasswing, Apache-2.0) as **content only** — mgh-sast never imports
vvaharness at runtime. Files are generated by
`tools/extract_prompts.py` (stdlib `ast`, verbatim) unless noted.

Fidelity levels: **verbatim** (byte-identical), **verbatim (composition)**
(assembled from verbatim parts in source order), **light-adapt** (Python
wrapping/interpolation removed, semantics unchanged), **rewrite** (noted with
rationale).

## Must-replicate workflow details (effectiveness-critical)

1. **s4 SYSTEM is byte-stable** across all calls; per-chunk language/specialist
   guidance lives in the USER prompt so prompt-cache markers hit every call.
2. **s6 majority-vote FP suppression** on backends that accept temperature.
3. **severity = CVSS 3.1 base-score band** (Critical 9.0–10 / High 7.0–8.9 /
   Medium 4.0–6.9 / Low 0.1–3.9 / Info = no demonstrated path). Label can never
   disagree with the vector.
4. **s5 deterministic pre-filter** is the primary FP defense.

## Regeneration

```bash
py tools/extract_prompts.py --repo-root . --vvaharness ./vvaharness --out ./mgh-sast/core/prompts
```
